# fedora-install-helper

My personal script to customize my Fedora install.

## Instructions

Boot your system using the Fedora 44 netinstall ISO in UEFI mode.

`Ctrl + Alt + F2` to switch to terminal.

To run the script:

``` bash
curl -sL https://raw.githubusercontent.com/Beck-D/fedora-install-helper/main/fedora-install-helper.sh
chmod +x fedora-install-helper.sh
./fedora-install-helper.sh
```

Follow the directions in the script and continue with the installation. Once completed, switch back to the terminal and run again.

``` bash
./fedora-install-helper.sh
```
